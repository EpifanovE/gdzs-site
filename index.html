Расчеты СИЗОД ГДЗС

Приложение-калькулятор предназначено для помощи при проведении расчетов расхода воздуха СИЗОД ГДЗС. Используется при решении задач и на посту безопасности при проведении учений и занятий.

Основные возможности

  • расчет для обычных и сложных условий работы звена ГДЗС;
  • режимы «очаг найден» и «очаг не найден»;
  • возможность устанавливать любое количество газодымозащитников;
  • поддержка ДАСК и ДАСВ;
  • для работы приложения не требуется подключение к интернету;
  • все функции доступны совершенно бесплатно.

Настройки параметров расчета

В приложении предусмотрены настройки параметров дыхательного аппарата и параметров расчета, необходимые для получения точного результата.

  • вместимость баллона;
  • количество баллонов;
  • давление воздуха (кислорода), необходимое для устойчивой работы редуктора;
  • средний расход воздуха (кислорода);
  • коэффициент сжимаемости воздуха;
  • давление срабатывания сигнального устройства;
  • учитывать ли сигнальное устройство при расчетах;
  • ведутся ли расчеты для ДАСК;
  • нужно ли округлять давление.

Решение в кратком и развернутом виде

При работе на учениях и занятиях важно получить быстрый результат. При этом, как правило, не нужны подробные решения задачи. Для этого калькулятор сначала показывает готовые ответы.

Напротив, при решении задач, необходимо предоставить подробное решение с ответом. Для этой цели приложение предоставляет детализированный результат с необходимыми формулами.

Заполнение журнала

Калькулятов может существенно облегчить заполнение журнал учета времени пребывания звеньев газодымозащитной службы (ГДЗС) в непригодной для дыхания среде. Все колонки журнала формируются исходя из проведенных расчетов. Также существует возможность автоматического вычисления давления в баллонах при выходе.
