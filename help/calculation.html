Заполнение данных

Установка условий работы звена ГДЗС (1)

Данный переключатель отвечает за то, какие условия работы будут применяться в расчетах. Если включено, то будут установлены сложные условия.

Режимы «очаг найден» и «очаг не найден» (2)

Переключение режимов «очаг найден» и «очаг не найден». Если включено, то будет применяться «очаг найден». В зависимости от выбранного варианта расчеты будут производиться по различным формулам.

Время включения в СИЗОД (3)

Время в часах и минутах, когда звено включилось в дыхательные аппараты. Возможны расчеты без указания этого параметра, в этом случае будут недоступны некоторые результаты, которые зависят от него, например ожидаемое время возвращения.

Время у очага (4)

Время в часах и минутах, при котором звено достигло очага пожара. Этот параметр доступен при включенном режиме «очаг найден». Можно не устанавливать даже если очаг пожара обнаружен, в таком случае в результатах будут отсутствовать некоторые данные, например время подачи сигнала о возвращении.

Давление в баллонах

В приложении возможна установка любого числа газодымозащитников. По умолчанию звено состоит из 3 человек. Для добавления еще одного предназначена кнопка 5. Для удаления необходимо нажать на кнопку удаления рядом с давлением соответствующего газодымозащитника.

Далее следуют поля для установки давления в баллонах участников звена. Каждая строка состоит из: давления при включении, давления у очага и кнопки удаления участника. Давление у очага необходимо указывать только в режиме «очаг найден».

Сброс данных (9)

При нажатии на кнопку 9 произойдет сброс данных, введенных на этом экране. Все поля примут первоначальное состояние.

Результы расчета

Кратко

При нажатии на кнопку «Рассчитать» (7) приложение произведет необходимые вычисления и покажет экран с краткими результатами. На этом экране представлены только ответы, которые могут быть вычислены исходя их введенных данных. Для получения подробной информации по расчету нужно нажать на соответствующий пункт.

Подробно

На экране подробной информации по расчету вначале указываются формулы, по которым производились вычисления. Затем идет решение с подстановкой данных, указанных ранее на главном экране приложения, их настроек, а также вычисленные системой. В самом конце приводится список с расшифровкой параметров формул, применяемых при текущем вычислении.
