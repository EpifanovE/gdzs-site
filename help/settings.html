Приложение «Расчеты СИЗОД ГДЗС» содержит необходимые настройки проведения расчетов. Большинство параметров следует брать из руководства по эксплуатации используемого СИЗОД. При настройке необходимо учитывать требования, предъявляемые в вашем подразделении.

Вместимость баллона

Вместимость одного баллона со сжатым воздухом или с кислородом в литрах, используемого в дыхательном аппарате. Указывайте здесь тот объем, который требуют от вас в вашем подразделении, т.к. в некоторых случаях допускается округление до целого числа, а в некоторых нет. Например: 7л и 6.8л. По умолчанию установлено значение 6.8л.

Количество баллонов

Количество баллонов, установленных на один дыхательный аппарат. Как правило, устанавливается 1 или 2 баллона.

Давление воздуха (кислорода), необходимое для устойчивой работы редуктора

Это значение нужно брать из инструкции по эксплуатации используемого СИОЗОД. Указывается в кгc/см². По умолчанию установлено значение 10 кгc/см², которое подходит для многих ДАСВ.

Средний расход воздуха (кислорода)

Значение среднего расхода воздуха в литрах в минуту, используемого при расчетах. Следуя «Методическим указаниям по проведению расчетов параметров работы в средствах индивидуальной защиты органов дыхания и зрения» от 2013 года, нужно указывать 40 л/м для ДАСВ и 2 л/м для ДАСК. По умолчанию установлено 40 л/м.

Коэффициент сжимаемости воздуха

Используется при проведении расчетов расхода воздуха ДАСВ. При использовании ДАСК не учитывается. По умолчанию установлено значение 1.1.

Давление срабатывания сигнального устройства

Давление, при котором срабатывает сигнальное устройство в кгc/см². Берется из руководства по эксплуатации СИЗОД. По умолчанию установлено 55 кгc/см².

Учитывать СУ

Если эта настройка активна, то при проведении расчетов, в случае если давление контроля выхода окажется меньше, чем давление срабатывания сигнального устройства, то в окончательном ответе будет использоваться давление срабатывания СУ вместо давления контроля выхода.

Расчеты кислорода

При активном режиме это настройки будут использоваться формулы для ДАСК. При этом также необходимо установить параметры используемого аппарата в соответствующих настройках.

Округление давления

Следует ли округлять полученные результаты при вычислении давления.
